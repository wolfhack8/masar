نبذة عن المشروع (150 كلمة)

تُعد منصة "مسار" مشروعاً ابتكارياً يهدف إلى إحداث نقلة نوعية في قطاع التعليم العالي من خلال دمج تقنيات الذكاء الاصطناعي لتوفير تجربة تعلم مخصصة لكل طالب. تنطلق فكرة المشروع من معالجة الفجوة المهارية بين المخرجات الأكاديمية ومتطلبات سوق العمل المتغيرة، خاصة في منطقة عسير. تعتمد المنصة على تحليل دقيق لمستوى الطالب لتحديد نقاط القوة والضعف، ومن ثم بناء مسارات تعليمية ذكية تتكيف مع وتيرة تعلمه وأهدافه المهنية. كما تضم المنصة مساعداً ذكياً يعمل بنماذج اللغة الكبيرة (LLMs) للإجابة الفورية على الاستفسارات، ونظام تحفيز يعزز التفاعل. يسعى "مسار" إلى تحقيق الاستدامة من خلال تقليل الهدر التعليمي والتنبؤ الاستباقي بحالات التعثر الدراسي، مما يدعم جودة المخرجات الوطنية بما يتوافق مع رؤية المملكة 2030. يستهدف المشروع الطلاب، الجامعات، وجهات التوظيف، لخلق منظومة تعليمية متكاملة تربط المنهج الأكاديمي بالمهارات التطبيقية والمحاكاة الواقعية لبيئات العمل.

وصف جذاب ومُلهم (200 كلمة)

مسار: حيث يرسم الذكاء الاصطناعي مستقبل طموحك الأصيل!

هل تساءلت يوماً لماذا يسلك الجميع الطريق الأكاديمي ذاته بينما يمتلك كل طالب شغفاً وبصمة فريدة؟ في عصر الثورة الرقمية المتسارعة، لم يعد التعليم التقليدي الموحد كافياً لصناعة التميز؛ ومن هنا ولدت منصة "مسار" لتكون المنارة التقنية الذكية التي تعيد تشكيل خارطة التعليم العالي، وتجسر بفخر تلك الفجوة المهارية بين مقاعد الدراسة ومتطلبات سوق العمل المتجدد.

"مسار" ليست مجرد منصة رقمية عابرة، بل هي منظومة ريادية مدعومة بأحدث خوارزميات الذكاء الاصطناعي، تعمل كشريك استراتيجي يحلل قدراتك بدقة، ويرسم لك رحلة تعلم مخصصة بالكامل تتكيف مع إيقاعك وأهدافك المهنية. وبفضل تقنيات المساعد الذكي التفاعلي المتاح على مدار الساعة، ومحاكاة الواقع الافتراضي، تنقلك المنصة إلى قلب بيئات العمل الحقيقية لتتخذ قراراتك المصيرية بثقة واقتدار.

نحن نتجاوز التلقين لنصنع التمكين؛ فمن خلال التنبؤ الاستباقي بالتعثر الدراسي ونظام التحفيز القائم على التحديات، نضمن استدامة رحلتك المعرفية وتقليل الهدر التعليمي. إن "مسار" هي البوابة الذهبية المتوافقة مع رؤية المملكة 2030 لتأهيل جيل وطني مبتكر ومستعد لقيادة المستقبل وإضاءة تطلعات الوطن بروح ريادية ملهمة.

وصف المشروع الشامل (1000 حرف)

منصة "مسار" هي مشروع تقني ريادي يهدف لإحداث ثورة في منظومة التعليم العالي عبر توظيف تقنيات الذكاء الاصطناعي لتوفير تجربة تعلم مخصصة بالكامل لكل طالب. تنطلق الفكرة من معالجة تحدٍ وطني جوهري يكمن في الفجوة المهارية بين المخرجات الأكاديمية ومتطلبات سوق العمل المتسارعة، مستهدفةً رفع جودة التعليم في منطقة عسير والمملكة تماشياً مع رؤية 2030.

تتميز المنصة بتقديم تحليل دقيق واستباقي لمستويات الطلاب، مما يتيح التنبؤ بالتعثر الدراسي قبل حدوثه ويقلل الهدر التعليمي والمادي للمؤسسات. توفر "مسار" مسارات تعليمية مرنة تتكيف مع إيقاع وأهداف كل متعلم، مدعومة بمساعد ذكي تفاعلي يعتمد على نماذج اللغة الكبيرة (LLMs) للإجابة الفورية، بجانب تقنيات المحاكاة الواقعية لبيئات العمل لتمكين الطلاب من اتخاذ قراراتهم المهنية بثقة.

يستهدف المشروع خلق منظومة متكاملة تربط الطلاب والجامعات بقطاع الأعمال، مما يضمن بناء مجتمع معرفي مستدام يرفد الوطن بكفاءات جاهزة ومؤهلة لقيادة المستقبل. "مسار" ليست مجرد منصة، بل هي البوصلة الذكية نحو تميزك الأكاديمي والمهني الصُنع لأجلك وبك.

دليل التثبيت ورفع المشروع (Installation & GitHub)

هذا القسم مخصص للمطورين ولجان التحكيم لتشغيل منصة "مسار" على أجهزتهم المحلية.

1. تثبيت Git على جهازك (إذا لم يكن مثبتاً):

  • نظام Windows: قم بتحميله من Git for Windows واتبع خطوات التثبيت الافتراضية.
  • نظام Mac: افتح موجه الأوامر (Terminal) واكتب: brew install git
  • نظام Linux: افتح موجه الأوامر واكتب: sudo apt-get install git

2. إعداد الحساب وربطه (Configuration):

# قم بإعداد اسمك وبريدك الإلكتروني المرتبط بحساب GitHub
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

3. استنساخ المشروع وتشغيله محلياً (Clone & Run):

# 1. استنساخ المستودع (Clone the repository)
git clone https://github.com/your-username/masaar.git

# 2. الانتقال إلى مجلد المشروع
cd masaar

# 3. تثبيت الاعتماديات الخاصة بالواجهة الأمامية (Frontend)
npm install

# 4. تثبيت الاعتماديات الخاصة بالذكاء الاصطناعي/الخلفية (Backend)
pip install -r requirements.txt

# 5. تشغيل المشروع
npm start
